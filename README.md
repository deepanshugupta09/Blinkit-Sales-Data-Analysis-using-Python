Blinkit Sales Data Analysis using Python
Project Overview

This project analyzes Blinkit grocery sales data using Python to extract meaningful business insights. The analysis includes data cleaning, exploratory data analysis (EDA), and visualization to understand sales patterns across outlet types, item categories, and locations.

Tools & Technologies Used

Python

Pandas – Data Cleaning & Analysis

NumPy – Numerical Operations

Matplotlib – Data Visualization

Seaborn – Statistical Visualization

Jupyter Notebook – Analysis Environment

Excel Dataset

Project Workflow
1. Data Import

The dataset was imported from an Excel file into a pandas DataFrame.

2. Data Exploration

Initial exploration included:

Viewing sample data

Checking dataset size

Understanding column information

Analyzing data types

3. Data Cleaning

Cleaning steps included:

Handling missing values

Correcting inconsistent data

Preparing data for analysis

4. Business Requirements

The project focuses on analyzing Blinkit sales performance based on:

Outlet characteristics

Product categories

Fat content

Outlet location

Establishment year

KPI Requirements

Key performance indicators analyzed in the project include:

Total Sales

Average Sales

Total Number of Items

Average Rating

Data Visualizations

Several charts were created using Python visualization libraries:

Total Sales by Fat Content

Total Sales by Item Type

Fat Content by Outlet for Total Sales

Total Sales by Outlet Establishment Year

Sales by Outlet Size

Sales by Outlet Location

Key Insights

Tier 3 outlets contribute the highest overall sales.

Medium-sized outlets generate the best performance.

Certain item types dominate the sales distribution.

Outlet establishment trends show how newer outlets impact revenue.
